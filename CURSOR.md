# Using this repo with Cursor

The 6-day cycle is a planning discipline more than a code-generation one. The Cursor rule helps the AI sequence work in cycle-friendly chunks.

## In this repository

1. Open the folder in Cursor.
2. The rule [`.cursor/rules/six-day-cycle.mdc`](.cursor/rules/six-day-cycle.mdc) is committed with `alwaysApply: false`.
3. Confirm under **Settings → Rules**.

## Use the same discipline elsewhere

**Cursor**: Copy `.cursor/rules/six-day-cycle.mdc` into that project's `.cursor/rules/`.

**Other AI tools**: Copy [`CLAUDE.md`](CLAUDE.md) to the project root.
