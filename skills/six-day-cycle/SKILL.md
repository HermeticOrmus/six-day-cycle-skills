# Six-Day Cycle

## Days

1. Ideate & Research (Mentalism)
2. Design & Plan (Correspondence)
3-4. Build (Vibration)
5. Test & Polish (Cause-Effect)
6. Launch (Gender)
Rest. (Rhythm) — sacred, non-optional

## Scales

1-hour, 3-day, 6-day, 2-week — pick what fits. Rest scales linearly only as a percentage; absolute rest can't shrink below a recoverable minimum.

## Anti-patterns

- 7-day weeks
- Borrowing rest day
- Stacking cycles
- All days as build days
